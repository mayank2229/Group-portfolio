# 👥 Group Portfolio - DevOps CI/CD Deployment Project

This project demonstrates the deployment of a **Group Portfolio Web Application** using a fully automated **CI/CD pipeline** with modern DevOps tools.

---

## 🌐 Project Overview

A collaborative **portfolio website** built as a group project and deployed using DevOps practices.

🔗 **GitHub Repository:** https://github.com/mayank2229/Group-portfolio

---

## ⚙️ DevOps Implementation

This project focuses on automating the deployment process using CI/CD:

### 🔁 CI/CD Pipeline Workflow

1. Developers push code to GitHub
2. GitHub Webhooks trigger Jenkins automatically
3. Jenkins pulls the latest code
4. Docker builds the application image
5. Container is deployed on AWS EC2
6. Application is updated automatically on every commit
